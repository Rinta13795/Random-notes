**怎么创建虚拟环境？像你这样的完整步骤：
bash

```bash
# 1. 进入项目文件夹
cd ~/my-project

# 2. 创建虚拟环境（名字通常叫 venv）
python3 -m venv venv

# 3. 激活虚拟环境
source venv/bin/activate

# 4. 安装需要的包
pip3 install 包名

# 5. 退出虚拟环境（用完后）
deactivate
```

**VS Code 配置**（让它自动激活）：

- `Cmd + Shift + P` → `Python: Select Interpreter` → 选择 `./venv/bin/python`
